# XiaoTian v0.1.0 更新日志

> 发布日期: 2025-03-02

## 核心工作

本版本为 XiaoTian 的初始版本，完成了基础项目设置与架构设计。

### 项目初始化

- 建立了基本的项目结构和目录组织
- 配置了开发环境与工具链
- 设置了基本的构建系统与依赖管理
- 添加了必要的项目文档和许可证

### 数据模型设计

- 实现了 `Repository` 数据模型，用于表示 GitHub 仓库
- 设计了 `Subscription` 初始模型，用于管理订阅关系
- 创建了基本的错误处理机制和自定义错误类型

### 存储层实现

- 设计并实现了存储特性（trait）接口，包括：
  - `Storage` - 基础存储接口
  - `RepositoryStorage` - 仓库存储操作
  - `SubscriptionStorage` - 订阅存储操作
- 实现了基于内存的存储实现 `MemoryStorage`，支持：
  - 存储和检索仓库信息
  - 管理订阅关系
  - 基本的 CRUD 操作

### 交互界面

- 实现了基础的命令行 REPL (Read-Eval-Print Loop) 界面
- 支持基本命令：
  - `help` - 显示帮助信息
  - `add` - 添加仓库
  - `list` - 列出仓库
  - `show` - 显示仓库详情
  - `delete` - 删除仓库
  - `clear` - 清空数据
- 实现了命令解析与错误处理逻辑
- 添加了彩色输出以提升用户体验

## 技术栈

- Rust 语言
- Tokio 异步运行时
- DashMap 用于并发安全的内存存储
- RustyLine 用于命令行界面
- Serde 用于序列化/反序列化
- Colored 用于终端彩色输出

## 已知问题

- 仅支持内存存储，重启后数据会丢失
- 尚未实现 GitHub API 集成
- 缺少订阅管理与更新检查功能

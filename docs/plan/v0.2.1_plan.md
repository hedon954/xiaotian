# XiaoTian v0.2.1 计划

## 概述

v0.2.1 版本将专注于将 GitHub 集成功能应用到 REPL 界面中，使用户能够通过命令行方便地获取和管理 GitHub 仓库更新。

## 功能点

1. **扩展 REPL 命令**

   - 添加 `fetch` 命令，允许用户获取订阅的最新更新
   - 添加 `config` 命令，允许用户设置 GitHub 令牌和其他配置
   - 扩展 `show` 命令以支持显示最近的更新

2. **实现 GitHub 令牌配置**

   - 允许用户通过 REPL 设置 GitHub 令牌
   - 在应用启动时加载令牌
   - 在创建 SourceFactory 时使用令牌

3. **更新获取与展示**

   - 实现从指定订阅获取更新的功能
   - 按类型（提交、问题、拉取请求、发布）分组显示更新
   - 支持过滤和限制显示的更新数量

4. **持久化**
   - 确保配置在重启后仍然可用
   - 可选地存储获取的更新以供后续查看

## 实现计划

1. 扩展 CommandHandler 以支持新命令
2. 创建配置存储和管理逻辑
3. 实现更新获取和显示逻辑
4. 更新帮助文档和示例
5. 测试新功能

## 后续规划

成功实现 v0.2.1 后，我们将为 v0.3.0 版本准备更高级的功能，如通知系统、更新报告生成和支持更多内容源。
